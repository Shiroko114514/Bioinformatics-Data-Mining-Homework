# 三项作业总结报告（Task1-Task3）

## 一、总体目的与目标

本报告以真核基因剪接位点预测为研究对象，围绕三类代表性方法进行实现与比较分析，核心目标如下：

1. 构建覆盖数据预处理、样本构造、模型训练、序列扫描与性能评估的端到端流程。
2. 系统比较生成式方法与判别式方法在剪接位点识别任务中的建模特性与性能差异。
3. 形成具备可复现性的代码框架与命令行使用方案，支持训练、模型持久化与新序列预测。

## 二、Task1：WMM/WAM 方法（加权矩阵模型）

### 1. 目的目标

1. 实现基于位点统计规律的剪接位点生成式建模。
2. 通过 WMM 与 WAM 对照实验，分析一阶依赖建模相对于独立位点假设的增益。

### 2. 具体内容

1. 从训练集与测试集提取 donor 位点窗口序列。
2. 构建正负样本并完成序列打分与分类判别。
3. 输出 Sensitivity、Specificity、Precision、F1、MCC、AUC 等指标并开展模型对比。

### 3. 方法

1. WMM：假设各位置条件独立，基于位置频率矩阵计算对数似然比。
2. WAM：引入相邻碱基依赖关系，以条件概率刻画局部上下文信息。
3. Dependency-WAM：通过依赖图增强的WAM，进一步建模位置之间的更复杂依赖关系。
4. 采用阈值决策将连续得分映射为二分类输出。

### 4. 编程实现

1. 代码拆分为数据读写、模型、扫描、预测与主程序等模块。
2. 支持 `--site donor` 和 `--site acceptor`，默认为 donor。
3. 训练和测试数据路径基于仓库位置自动解析，不依赖硬编码绝对路径。
4. 当真实数据不可用时，自动生成合成数据作为回退。
5. `main.py` demo 会自动生成 ROC 曲线图 `roc_curves.svg`。
6. ROC 绘图逻辑单独拆分到 `roc_plot.py`，支持复用。
7. 支持模型训练后的保存与复用，并支持基因组序列滑窗扫描。

### 5. 测试

1. 在给定训练/测试数据集上完成训练与独立测试。
2. 通过 WMM 与 WAM 的指标差异验证方法有效性与改进方向。
3. 运行 `python main.py --demo` 可自动生成 `roc_curves.svg`，展示 WMM、WAM、Dependency-WAM 三种方法的 ROC 曲线比较。

### 6. 结果

1. Task1 完成了生成式基线系统搭建，现包含 WMM、WAM、Dependency-WAM 三种方法对比。
2. 实验上 WAM 通常在 donor 识别任务中优于 WMM，体现一阶依赖特征的有效性。
3. Dependency-WAM 进一步展现了更复杂依赖结构的建模潜力。

## 三、Task2：贝叶斯网络方法

### 1. 目的目标

1. 在 WAM 基础上进一步放宽结构假设，学习位置之间更一般的依赖关系。
2. 验证贝叶斯网络在剪接位点识别任务中的表达能力与泛化能力。

### 2. 具体内容

1. 使用贝叶斯网络对位点间依赖关系进行显式建模。
2. 实现结构学习策略（如 Chow-Liu 及可扩展结构搜索策略）。
3. 与 WMM/WAM 在统一评估协议下进行比较。

### 3. 方法

1. 采用图模型表示碱基变量之间的依赖结构。
2. 支持多种结构学习策略：
   - Chain：假设相邻位置形成链式结构。
   - Chow-Liu：基于互信息最大化生成树结构。
   - EBN（Extended Bayesian Network）：基于 χ² 测试学习多父节点结构。
3. 通过结构学习与参数学习建立正负类概率模型。
4. 基于对数似然比或后验得分完成分类与扫描。

### 4. 编程实现

1. 代码按 data/model/eval/main 分层组织，模块边界清晰。
2. 支持命令行参数：
   - `--site`：donor 或 acceptor（默认 donor）
   - `--threshold`：分数阈值（默认 0.0）
   - `--window`：窗口大小（9 或 23，自动选择）
   - `--structure`：结构类型（chain、chow-liu 或 ebn，默认 chow-liu）
   - `--max-parents`：EBN 中每个节点最多父节点数（默认 2）
   - `--chi2-threshold`：EBN 依赖边的 χ² 阈值（默认 6.0）
   - `--no-real-data`：使用合成数据而非真实集
3. 数据路径自动从仓库 `Training and testing datasets/` 追踪。
4. Demo 运行时会自动生成 ROC 曲线图 `roc_curves.svg`，包含 WMM、WAM、BN Chow-Liu 和 BN EBN(p=2) 的对比。

### 5. 测试

1. 在统一数据分割下完成公平比较。
2. 输出标准分类指标与模型对比结果。
3. 运行 `python3 splice_main.py --demo` 可自动生成 `roc_curves.svg`，展示 BN Chow-Liu 和 BN EBN(p=2) 两种贝叶斯网络结构的 ROC 曲线。

### 6. 结果

1. Task2 成功实现贝叶斯网络方法，满足“Task2 使用贝叶斯网络”的任务要求。
2. 支持多种网络结构（Chain、Chow-Liu、EBN），展现灵活的依赖关系建模能力。
3. 在可解释性层面，可观察到非相邻位点依赖关系与多父节点结构带来的建模收益。
4. EBN 结构学习通过 χ² 统计检验自动发现重要依赖关系。

## 四、Task3：支持向量机方法（SVM）

### 1. 目的目标

1. 构建判别式模型，直接学习剪接位点与非位点之间的分类边界。
2. 实现完整的特征工程、模型训练、评估与预测流程。
3. 与生成式模型形成方法学对照，完成三任务闭环分析。

### 2. 具体内容

1. 实现特征工程模块：one-hot、kmer2、kmer3、dinuc_pos、pwm 及其组合特征。
2. 实现 SVM 训练、交叉验证、线性核特征权重分析及全基因组滑窗扫描。
3. 完成训练保存脚本与预测脚本，形成独立可运行流程。
4. 支持特征消融实验、核函数比较与四方模型对比（WMM/WAM/BN/SVM）。

### 3. 方法

1. 分类器采用 SVC，支持线性核、RBF 核与多项式核。
2. 建模流程为 FeatureExtractor + StandardScaler + SVM Pipeline。
3. 评估指标包括 AUC、MCC、F1，并开展特征消融与核函数比较实验。

### 4. 编程实现

1. 环境依赖：Python 3.13+、numpy、scikit-learn。
2. 代码已模块化拆分为 utils、data、features、model、eval、main。
3. 提供多套脚本与入口：
   - `wrapper.py`：训练模型并保存到 `svm_splice_site.pkl`
   - `predict.py`：加载模型并对输入 DNA 序列进行扫描预测
   - `splice_main.py`：完整 demo 包括特征消融、核函数比较、四方模型对比、线性 SVM 重要特征分析、5折交叉验证、genome scan
4. ROC 绘图逻辑单独拆分到 `roc_plot.py`，输出 SVG 格式图文件。
5. Demo 运行时会自动生成 `roc_curves.svg`，包含多模型 ROC 曲线对比。
6. 通过统一接口保证训练与预测链路可维护、可复现。

### 5. 测试与评估

1. 支持训练集/测试集分离评估。
2. 支持 5-fold 交叉验证（每折内重新拟合特征提取器，避免 PWM 特征泄漏）。
3. 支持输出扫描位点位置、得分与窗口上下文。
4. 支持特征消融实验，识别各特征组件对性能的贡献。
5. 支持多核函数比较（线性、RBF、多项式）。
6. 运行 `python3 splice_main.py` 可自动生成 `roc_curves.svg`，展示 SVM (RBF) 核函数的 ROC 曲线。

### 6. 结果

1. Task3 采用标准支持向量机实现，满足“Task3 使用支持向量机”的任务要求。
2. 在方法层面与 Task2（贝叶斯网络）形成清晰区分，体现判别式与生成式的对照。
3. 通过特征消融与核函数对比，展现 SVM 的灵活塑造能力。
4. 训练与预测链路完整，支持完整演示、特征分析和四方模型对比。
5. 基于 scikit-learn 的模块化实现保证了代码的可维护性与可扩展性。

### 4. 三种方法对比表

| 方法 | 主要优点 | 主要缺点 | 时间复杂度（训练/推断） | 可解释性 |
| --- | --- | --- | --- | --- |
| WMM/WAM/Dependency-WAM | 实现简单，参数含义直观；对小样本较稳健；便于作为基线；Dependency-WAM 通过依赖图进一步增强表达能力 | 对高阶依赖刻画有限，表达能力受结构假设约束；Dependency-WAM 结构固定，学习能力有界 | 训练近似 $O(N \cdot L)$ 到 $O(N \cdot L^2)$；推断近似 $O(L)$ | 高 |
| 贝叶斯网络（BN） | 可显式建模位点依赖关系；支持多种结构学习策略（Chain、Chow-Liu、EBN）；结构可解释性强；EBN 可自动发现多父节点关系 | 结构学习与参数学习成本较高；实现与调参复杂度较大；EBN χ² 阈值需要调整 | 训练约为 $O(N \cdot L^2)$ 到 $O(N \cdot L^3)$（依赖结构搜索策略和复杂度）；推断约 $O(L)$ | 高 |
| 支持向量机（SVM） | 判别能力强；可通过核函数处理非线性边界；在多特征组合下表现稳定；特征消融分析清晰 | 参数选择敏感；模型可解释性弱于概率图模型；大规模训练成本较高；核函数选择依赖经验 | 训练复杂度约为 $O(N^2)$ 到 $O(N^3)$（与核和求解器相关）；推断约 $O(n_{SV} \cdot d)$ | 中 |
